All authors presenting at IPAC'28 must submit a contribution paper (maximum 3 pages for posters/contributed orals, 5 pages for plenary/invited talks). Papers must adhere strictly to the **JACoW (Joint Accelerator Conferences Website)** style templates.

### LaTeX Preparation Instructions
JACoW provides a highly optimized LaTeX template environment which is the preferred format for submissions.

1. **Download Template**: Get the official JACoW LaTeX ZIP bundle from [JACoW Templates Repository](https://www.jacow.org/).
2. **Class File**: Ensure you are using the latest `jacow.cls` document class file.
3. **Margins & Font**:
   - *Paper Size*: A4 or US Letter (must compile exactly to page boundary limits).
   - *Fonts*: Uses Times/Times New Roman. Math symbols must be rendered using standard symbols to avoid font missing.
   - *Columns*: Strict two-column format (column width: 82mm, spacing: 5.3mm).
4. **Warning**: Do not modify class settings or page margins. Over-length papers will not be processed by the editorial team.

    {{< tab title="MS Word Author Templates" >}}
### MS Word Preparation Instructions
If you prepare your paper using Microsoft Word, please follow these guidelines to ensure proper formatting:

1. **Download Template**: Get the official JACoW MS Word DOTX file from the JACoW portal.
2. **Paragraph Styles**: Use the pre-configured styles built into the template (e.g., `JACoW_Paper_Title`, `JACoW_Author_List`, `JACoW_Body_Text`).
3. **Common Pitfalls**:
   - *Margins*: 20mm top, 20mm bottom, 20mm left, 20mm right.
   - *Hyphenation*: Enable automatic hyphenation to avoid excessive space in columns.
   - *Equations*: Use the native Math Equation editor rather than placing them as floating graphic boxes.
    {{< /tab >}}

    {{< tab title="Upload & PDF Verification" >}}
### Upload Procedures
All files must be uploaded before the paper deadline: **May 20, 2028**.

1. **Source Files**: You must upload your source file (`.tex` or `.docx`), all high-resolution figures (`.eps`, `.png`, `.jpg`), and the compiled high-quality PDF.
2. **PDF Requirements**:
   - Must have all fonts **strictly embedded** (subsetted).
   - Resolution of images must be at least 300 dpi.
   - No page numbers, headers, or footers should be present in the PDF.
3. **JACoW Cat Scan**: The online Indico submission will run an automated script (Cat Scan) to verify margins, font embedding, and outline boundaries. Any failing issues must be resolved by uploading revised drafts.
